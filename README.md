# Marked_Pk

## Overview


**Note:** This README is still rough and will be updated. All data is currently stored on glamdring, the Oxford astrophysics cluster. 

## Contents

- `Mark_Class.py`: Main class for marked statistics analysis, including Gaussian Process modeling and power spectrum calculations. (Rough around the edges)
- `Example_Notebook.ipynb`: Demonstrates how to load arrays and visualize data used in the project.
- `Reproduce Figure 3.ipynb`: Script to reproduce a key figure from the associated research, including advanced plotting and Gaussian Process regression.
